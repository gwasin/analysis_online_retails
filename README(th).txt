**การวิเคราะห์ข้อมูลการค้าปลีกออนไลน์**

รายละเอียด : โปรเจกต์นี้เป็นการวิเคราะห์ ข้อมูลการค้าปลีกออนไลน์ เพื่อหาแนวโน้มพฤติกรรมลูกค้า การขาย และโอกาสทางธุรกิจ

เป้าหมาย :	- วิเคราะห์พฤติกรรมการซื้อของลูกค้า

	- หาสินค้าและหมวดหมู่ที่ขายดีที่สุด

	- วิเคราะห์แนวโน้มยอดขายตามช่วงเวลา

	- แบ่งกลุ่มลูกค้าตามวงจรชีวิต

	- (ถ้ามี) พยากรณ์ยอดขายรายเดือนโดยใช้ time series

ชุดข้อมูล : - ที่มา: UCI / Kaggle

ชื่อไฟล์: OnlineRetail.csv

ข้อมูลประกอบด้วยธุรกรรมของร้านค้าออนไลน์ในสหราชอาณาจักรช่วงปี 2010–2011

เครื่องมือที่ใช้ : Python, SQL ,PowerBI

ขั้นตอนหลัก :  - การทำความสะอาดข้อมูล (ลบค่า missing, duplicate, แก้ชนิดข้อมูล)

	- การวิเคราะห์เบื้องต้น (EDA)

	- วิเคราะห์ RFM เพื่อแบ่งกลุ่มลูกค้า

	- วิเคราะห์ยอดขายตามช่วงเวลา

	- (ถ้ามี) การพยากรณ์ยอดขายในอนาคต
